# Quiz
A demo app for AppliedAI Consultings

# To run the app
1. ``` $ git clone https://github.com/nandini8/Quiz.git ```
2. ``` $ cd app```
3. ``` $ python scripts/main.py```

# Using docker
``` $ docker run nandini8/quiz ```

OR

1. ``` $ git clone https://github.com/nandini8/Quiz.git ```
2. ``` $ cd app ```
3. ``` $ docker build -t quiz . ```
4. ``` $ docker run quiz ```
